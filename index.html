Cara Kerja Simulasi Sistem Bot Terupdate:

1. Bot terhubung ke Web API *LostPotential*.
2. Setiap ada input lomba baru, bot membaca skrip dan melakukan *broadcast* otomatis ke nomor WhatsApp anggota.
3. Ketik kata kunci **!infolomba** atau **!caribakat** di nomor WA grup untuk memanggil database.
